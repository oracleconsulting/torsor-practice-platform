# RPGCC Logo Assets

Please save your logo files in this folder with the following names:

## Required Files:

1. **`rpgcc-logo.svg`** (or `.png`)
   - Main logo for light backgrounds (dark logo on transparent background)
   - Recommended: SVG format for best quality
   - If using PNG: minimum 400px width, transparent background

2. **`rpgcc-logo-white.svg`** (or `.png`)
   - Logo for dark backgrounds (white/light logo on transparent background)
   - Recommended: SVG format for best quality
   - If using PNG: minimum 400px width, transparent background

## File Location:
```
apps/client-portal/public/logos/
```

## Usage:
The logos will be accessible at:
- `/logos/rpgcc-logo.svg` (for light backgrounds)
- `/logos/rpgcc-logo-white.svg` (for dark backgrounds)

## Notes:
- SVG format is preferred for scalability
- PNG format is acceptable if SVG is not available
- Ensure transparent backgrounds for best results
- Logo should include the three colored dots if they're part of the brand
